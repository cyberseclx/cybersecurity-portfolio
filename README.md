# Cybersecurity Portfolio — Rohit Dangar

This repository documents my cybersecurity learning journey through legal, authorized labs and self-built projects.

## Areas Covered

* Networking fundamentals
* Linux administration and security
* Windows and Active Directory fundamentals
* SOC alert triage and incident reporting
* Web application security
* Python and Bash automation
* Vulnerability assessment and reporting

## Lab Platforms

* TryHackMe
* Hack The Box Academy
* PortSwigger Web Security Academy
* Local Kali Linux virtual machine
* Intentionally vulnerable lab environments

## Repository Structure

| Folder                        | Purpose                                                                   |
| ----------------------------- | ------------------------------------------------------------------------- |
| `01-networking`               | TCP/IP, subnetting, DNS, DHCP, ARP, Wireshark and packet-analysis notes   |
| `02-linux`                    | Linux commands, permissions, services, logs and security learning notes   |
| `03-windows-active-directory` | Windows administration, PowerShell and Active Directory fundamentals      |
| `04-soc-analyst`              | Alert triage, phishing analysis, incident notes and investigation reports |
| `05-web-security`             | HTTP, Burp Suite, OWASP concepts and legal web-security lab reports       |
| `06-scripts`                  | Bash and Python scripts created during learning                           |
| `07-reports`                  | Professional-style vulnerability-assessment and incident-response reports |
| `08-cheatsheets`              | Personal command references and study notes                               |

## Security and Ethics

All work in this repository is limited to authorized training platforms, local lab environments and intentionally vulnerable targets. No real systems, private data, credentials, company information or unauthorized testing is included.
